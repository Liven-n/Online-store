Handmade Jewelry & Gifts Store
A responsive multi-page e-commerce website for handmade accessories, developed as a pet project to demonstrate frontend development skills using HTML5, CSS3, and JavaScript.

Overview
This website allows users to browse products across three categories — rings, pins, and greeting cards — view promotional discounts, leave feedback, and submit purchase orders. The project features a fully responsive design, adaptive product grid, and interactive user interface.

Key Features
Responsive Layout:
Mobile-first design with media query breakpoints (450px, 768px, 1000px)
Product grid automatically adjusts from 1 to 4 columns based on screen size

Product Catalog:
Three category pages: rings, pins, greeting cards
Product cards with image, description, old price (strikethrough), and sale price
Dedicated promotions page with discounted items

User Interaction:
Customer reviews section with avatar images and testimonials
Feedback form for submitting comments
Order placement form with required field validation
Interactive "Add to cart" buttons with JavaScript alerts

Navigation:
Multi-page structure (10+ HTML pages)
Main menu, catalog menu, and search bar interface
Smooth navigation between all sections

UI Components:
Sticky header with store name
Category gallery on homepage with image links
Order confirmation page
"About" section and contact information
